    How the strategies differ

    • Separate Chaining: Keep a mini list in each bucket.
    • Linear Probing: Move right one step at a time until an empty slot is found.
    • Quadratic Probing: Jump using squares (1², 2², 3², ...) for a wider search.
    • Double Hashing: Use a second hash function to jump by larger, consistent steps.

    Open-addressing tables automatically resize to nearby prime sizes to keep probes short. Separate chaining rebalances when chains grow too long.
